# Smart-Dopefolio – Personal Portfolio Website

This repository contains a personal portfolio website developed using
HTML, CSS, JavaScript, and SASS.

The project is customized from an open-source portfolio template and
modified for academic and personal learning purposes.

## Project Description

Smart-Dopefolio is a clean and responsive personal portfolio website
created for college submission. The website presents personal details,
skills, projects, and contact information in a professional format.

This project is suitable for students and beginners who want to
showcase their profile using basic frontend web technologies.

---

## Customizations & Work Done

- Replaced all template content with personal information  
- Modified color theme and UI styling  
- Added personal projects and descriptions  
- Updated footer, contact section, and social links  
- Customized layout structure as per academic requirements

---

## How to Run the Project

1. Download or clone the repository  
2. Open the project folder in VS Code  
3. Open `index.html` in any web browser  
4. (Optional) Use Live Server for better experience
